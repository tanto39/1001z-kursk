Документация для веб-службы Автопитер можно посмотреть на странице: http://service.autopiter.ru/price.asmx.
Веб-служба работает по протоколу SOAP(http://ru.wikipedia.org/wiki/SOAP).

ВАЖНО!
Если ваша версия выше PHP 5.0.1, то вы можете воспользоваться встроенным классом
SoapClient, иначе необходимо использовать библиотеку NuSoap или др. В данном примере используются
встроенные возможности PHP. http://php.net/manual/en/book.soap.php

При вызове PHP класса аргумент ссылки указывается "http://service.autopiter.ru/price.asmx?WSDL", наличие "http://" и "?WSDL" обязательно.
